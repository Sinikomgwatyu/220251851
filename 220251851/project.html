Our Proposed Team Project

Project Title

Our project is titled Smart Campus Management System. The system will help students and staff manage schedules, assignments, announcements, and events efficiently in one platform.

Project Objectives

  • Develop a user-friendly web application.
  • Provide real-time notifications for events and deadlines.
  • Allow students to access academic resources easily.
  • Help lecturers manage classes and assignments digitally.

Technologies To Be Used

We plan to use HTML, CSS, JavaScript, PHP, and MySQL for the development of the project. The server will run on Apache (XAMPP).
